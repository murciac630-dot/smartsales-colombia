# 🚀 SmartSales: IA para el Sector Agrícola

**SmartSales** es un MVP (Producto Mínimo Viable) diseñado para predecir tendencias de precios en el mercado mayorista del Valle del Cauca.

## 🧠 El Modelo
Utiliza un algoritmo de **Regresión Lineal** para analizar el historial de precios y proyectar el valor del kilo para el día siguiente.

## 🛠️ Cómo ejecutarlo
Esta aplicación está diseñada para desplegarse en **Streamlit Cloud**.
1. Instala las dependencias: `pip install -r requirements.txt`
2. Ejecuta la app: `streamlit run app.py`

## ⚖️ Ética
Este proyecto busca reducir la asimetría de información entre productores y comerciantes, promoviendo un comercio justo en Colombia.
